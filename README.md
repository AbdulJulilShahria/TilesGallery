# Tiles Gallery

A simple gallery project with user authentication and category-based tile browsing.

## Features

- User Authentication
  - Sign Up
  - Login
  - Logout
- Browse tile collections
- Filter tiles by category
- View all categories
- Save selected/favorite tiles
- Responsive design for mobile and desktop

## Categories

- All Categories
- Wall Tiles
- Floor Tiles
- Bathroom Tiles
- Kitchen Tiles
- Outdoor Tiles

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Next.js

### Backend
- Node.js
- Express.js
- MongoDB
- Better Auth Authentication

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/tiles-gallery.git
cd tiles-gallery
```

### 2. Install Frontend Dependencies

```bash
cd client
npm install
```

### 3. Install Backend Dependencies

```bash
cd ../server
npm install
```

## Running the Project

### Start Backend Server

```bash
cd server
npm run dev
```

### Start Frontend Application

Open a new terminal:

```bash
cd client
npm run dev
```


## Usage

1. Register a new account.
2. Login using your credentials.
3. Browse the tile gallery.
4. Filter tiles by category.
5. Select and save favorite tiles.
6. View all categories or specific tile collections.
